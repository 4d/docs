### Description

Sets the selected text to appear darker and heavier.

**EXAMPLE**

This is normal text.
**This is bold text.**




### JSON Grammar

|Property|Data Type|Possible Values|
|---|:---:|---|
|fontWeight|text | "normal", "bold"|
