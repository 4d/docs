### Description

Specifies the source of the data.


### JSON Grammar

|Name|Data Type|Possible Values|
|:---|:---:|:---:|
|dataSource|string, or string array for hierarchical listbox column|A 4D variable, field name, or an arbitrary complex language expression.|
