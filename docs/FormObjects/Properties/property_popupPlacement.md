### Description

Allows displaying a symbol that appears as a triangle in the button, which indicates that there is a pop-up menu attached.	

**EXAMPLE**


### JSON Grammar

|Name|Data Type|Possible Values|
|:---:|:---:|---|
|popupPlacement|string |"none", "linked", "separated"|
