### Description

Mandatory. Designates the data type of the form object.



### JSON Grammar

|Name|Data Type|Possible Values|
|:---|:---:|:---:|
|type|string|"text", "rectangle", "groupBox", "tab", "line", "button", "checkbox", "radio", "dropdown", "combo", "webArea", "write", "subform", "plugin", "splitter", "buttonGrid", "progress", "ruler", "spinner", "stepper", "list", "pictureButton", "picturePopup", "listbox", "input", "view"|
