## Horizontal Line Color

Defines the color of the horizontal lines in a list box (gray by default).

#### JSON Grammar

|Name|Data Type|Possible Values|
|---|---|---|
|horizontalLineStroke| color|Any CSS value, "transparent", "automatic"|

#### Objects Supported

[List Box](listbox_overview.md)

#### Commands

[LISTBOX GET GRID COLORS](../commands-legacy/listbox-get-grid-colors.md) - [LISTBOX SET GRID COLORS](../commands-legacy/listbox-set-grid-colors.md)
