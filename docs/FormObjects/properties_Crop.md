## Rows

Sets the number of rows in a thumbnail table.

#### JSON Grammar

|Name|Data Type|Possible Values|
|:---|:---:|---|
|rowCount|integer|minimum: 1|


#### Objects Supported

|Category|Link| 
|---|---|
|Picture Button|[Picture](pictureButton_overview.md)|
|Button Grid|[Button Grid](buttonGrid_overview.md)|
