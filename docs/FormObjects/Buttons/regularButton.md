### Description

The Regular button style is a standard system button (*i.e.*, a rectangle with a descriptive label) which executes code when a user clicks on it.

By default, the Regular button style has a light gray background with a label in the center: 

![alt-text](assets/en/regularButton.en.png)

When the cursor hovers over the Regular button style, the border and background color change to demonstrate that it has the focus:

![alt-text](assets/en/regularButtonHover.en.png)
<p>

In addition to initiating code execution, the Regular button style mimics a mechanical button by quickly changing background color when being clicked:


![alt-text](assets/en/regularButton_clicked.en.png)



#### Selecting the style

The Regular button's style property can be set via:

* **Property List:** Select *Regular* from the **Button Style** drop-down list in the **Text and Picture** section.  <br>![alt-text](assets/en/regularButton_propertyList.en.png)


* **Code:** Define the property using JSON: <br>Example: ```{"style":"regular"} ```<br>
See the [Dynamic Forms](https://doc.4d.com/4Dv17R5/4D/17-R5/Dynamic-Forms.300-4163740.en.html#3692292) page for more information.
