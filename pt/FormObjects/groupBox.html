Área de grupo

Uma área de grupo é um objeto estático que permite montar visualmente vários objetos de formulário:

O nome de uma caixa de grupo é texto estático; você pode usar uma referência "localizável" como com qualquer rótulo 4D (veja Usando referências em texto estático e seção Arquitetura XLIFF em referência Desenho 4D.

Exemplo JSON:​

	"myGroup": {
		"type": "groupBox",	
		"title": "Employee Info"
		"left": 60,
		"top": 160,
		"width": 100,
		"height": 20
		}

Propriedades compatíveis​

Inferior - Class CSS - Fonte - Cor da fonte - Tamanho da fonte - Altura - Alinhamento horizontal - Dimensionamento horizontal - Itálico - Esquerda - Nome do objeto - Certo - Título - Topo - Tipo - Sublinhado - Dimensionamento vertical - Visibilidade - Largura
