OBJECT Get data source

OBJECT Get data source ( {* ;} objeto ) : Pointer

ParâmetroTipoDescrição
*Operador→Se especificado, objeto é um nome de objeto (cadeia) Se omitido, objeto é um campo ou uma variável
objetoany→Nome de objeto (se * for especificado) ou
Campo ou variável (se * for omitido)
ResultadoPointer←Ponteiro a fonte de dados atual do objeto

Descrição​

O comando OBJECT Get data source devolve a fonte de dados atual dos objetos designados pelos parâmetros objeto e * .

Pode definir a fonte de dados para um objeto em modo Desenho utilizando a Lista de propriedades, ou utilizando o comando OBJECT SET DATA SOURCE.

Passando o parâmetro opcional * indica que o parâmetro objeto é um nome de objeto (cadeia). Se não passar este parâmetro, indica que o parâmetro objeto é um campo ou uma variável. Neste caso, se passar uma referência de campo ou variável em lugar de uma cadeia (campo ou variável objeto unicamente).

Exemplo​

Dado um objeto combo box definido em um formulário:

Executa o seguinte código:

 $vPtr :=OBJECT Get data source(*;"vCombo")
  // $vPtr contém -> vCombo
