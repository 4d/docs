API (dataClass)

API REST para dataClass.

📄️ dataClass

Os nomes de dataclass podem ser utilizados diretamente nos pedidos REST para trabalhar com entidades e selecções de entidades, ou funções de classe da dataclass.

📄️ $asArray

Devolve o resultado de uma consulta num array (ou seja, uma coleção) em vez de um objecto JSON.

📄️ $atomic/$atOnce

Permite que as ações no pedido REST estejam numa transação. Se não houver erros, a transação é validada. Caso contrário, a transação é cancelada.

📄️ $attributes

Permite seleccionar los atributos relacionales a obtener de la dataclass (por ejemplo, Company(1)?$attributes=employees.lastname o Employee?$attributes=employer.name).

📄️ $binary

Pase "true" para guardar el BLOB como documento (también debe pasar $expand=)

📄️ $clean

Creates a new entity set from an existing entity set but without its deleted entities, if any (e.g. $clean=true)

📄️ $compute

Cálculo de atributos específicos (e., Employee/salary/?$compute=sum) o en el caso de un atributo Objeto (por ejemplo, Employee/objectAtt.property1/?$compute=sum)

📄️ $distinct

Devuelve los diferentes valores de un atributo específico en una colección (por ejemplo, Company/name?$filter="name=a*"&$distinct=true)

📄️ $entityset

Después de crear un conjunto de entidades mediante el uso de $method=entityset, puede utilizarlo posteriormente.

📄️ $expand

Expande uma imagem armazenada em um atributo Image (\*por exemplo, Employee(1)/photo?$imageformat=best&$expand=photo)

📄️ $filter

Permite consultar los datos de una clase de datos o de un método (p. ej., $filter="firstName!='' AND salary>30000")

📄️ $format

Defines a specific format to return data of a dataclass, (e.g., $format=looker)

📄️ $imageformat

Define qué formato de imagen utilizar para recuperar imágenes (por ejemplo, $imageformat=png)

📄️ $lock

Bloquea y desbloquea una entidad utilizando el mecanismo pesimista.

📄️ $method

Este parâmetro permite-lhe definir a operação a executar com a entidade ou a seleção de entidades devolvida.

📄️ $orderby

Ordena los datos devueltos por el atributo y el orden de clasificación definidos (por ejemplo, $orderby="lastName desc, salary asc")

📄️ $querypath

Devuelve la petición tal y como se ejecutó por 4D Server (\por ejemplo, \, $querypath=true)

📄️ $queryplan

Devuelve la petición tal y como se pasó a 4D Server (\por ejemplo, \, $queryplan=true)

📄️ $savedfilter

Guarda el filtro definido por $filter al crear un conjunto de entidades (por ejemplo, $savedfilter="")

📄️ $savedorderby

Guarda el filtro definido por $orderby al crear un conjunto de entidades (por ejemplo, $savedorderby="")

📄️ $skip

Inicia la entidad definida por este número en la colección (por ejemplo, $skip=10)

📄️ $timeout

Define el número de segundos para guardar un conjunto de entidades en la caché de 4D Server (por ejemplo, $timeout=1800)

📄️ $top/$limit

Limita el número de entidades a devolver (por ejemplo, $top=50)

📄️ $version

Número da versão da imagem
