Liste de propriétés JSON

Cette page fournit une liste complète de toutes les propriétés du formulaire, triées par leur nom JSON. Cliquez sur un nom de propriété pour accéder à sa description détaillée.

Dans le chapitre "Propriétés du formulaire", les propriétés sont triées en fonction de leurs noms et thèmes dans la liste des propriétés.

PropriétéDescriptionValeurs possibles
a
bottomMarginValeur de la marge verticale (en pixels)minimum : 0
d
destinationType de formulaire"detailScreen", "listScreen", "detailPrinter", "listPrinter"
e
entryOrderL'ordre dans lequel les objets actifs sont sélectionnés lorsque l'onglet ou la touche retour chariot est utilisé(e) dans un formulaire de saisieCollection de noms d'objets 4D Form
eventsListe de tous les événements sélectionnés pour l'objet ou le formulaireCollection de noms d'événements, ex : ["onClick","onDataChange"...].
f
formSizeAnchorNom de l'objet dont la position détermine la taille du formulaire. (longueur minimale : 1)Nom d'un objet 4D
h
heightHauteur du formulaireminimum : 0
i
inheritedFormDésigne le formulaire à hériterNom (chaîne) de la table ou du formulaire projet OU d'un chemin POSIX (chaîne) vers un fichier .json décrivant le formulaire OU un objet décrivant le formulaire
inheritedFormTableDésigne la table qu'un formulaire hérité utiliseraUn nom ou un numéro de table
m
markerBodyPosition du marqueur de détailminimum : 0
markerBreakPosition(s) du marqueur de ruptureminimum : 0
markerFooterPosition du marqueur de piedminimum : 0
markerHeaderPosition(s) du marqueur d'en-têteinteger minimum: 0; integer array minimum: 0
memorizeGeometryEnregistre les paramètres du formulaire lorsque la fenêtre du formulaire est ferméetrue, false

methodLe nom d'une méthode projet.Le nom d'une méthode projet existante

pageFormatobjectPropriétés d'impression disponibles

rightMarginValeur de la marge horizontale (en pixels)minimum : 0
s
sharedIndique si un formulaire peut être utilisé comme sous-formulairetrue, false

widthLargeur du formulaireminimum : 0
windowMaxHeightLa plus grande hauteur autorisée de la fenêtre de formulaireminimum : 0
windowMaxWidthLa plus grande largeur autorisée de la fenêtre de formulaireminimum : 0
windowMinHeightLa plus petite hauteur autorisée de la fenêtre de formulaireminimum : 0
windowMinWidthLa plus petite largeur autorisée de la fenêtre de formulaireminimum : 0
windowSizingXDimensionnement vertical de la fenêtre de formulaire"fixed", "variable"
windowSizingYDimensionnement horizontal de la fenêtre de formulaire"fixed", "variable"
