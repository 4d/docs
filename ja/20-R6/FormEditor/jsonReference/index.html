フォーム JSON プロパティリスト

この一覧は、すべてのフォームプロパティを JSON名順に並べてまとめたものです。 プロパティ名をクリックすると、詳細説明に移動します。

"フォームプロパティ" の章では、プロパティリストにおけるテーマと名称で各プロパティを紹介しています。

b - c - d - e - f - h - i - m - p - r - s - w

プロパティ説明とりうる値
b
bottomMargin垂直マージン値 (ピクセル単位)最小値: 0
c
colorSchemeフォームのカラースキーム"dark", "light"
cssフォームが使用する CSSファイル文字列、文字列のコレクション、または "path" と "media" プロパティを持つオブジェクトのコレクションとして提供される CSSファイルパス。
d
destinationフォームタイプ"detailScreen", "listScreen", "detailPrinter", "listPrinter"
e
entryOrder入力フォームで Tabキーや 改行キーが使用されたときに、アクティブオブジェクトが選択される順番。4Dフォームオブジェクト名のコレクション
eventsオブジェクトまたはフォームについて選択されているイベントのリストイベント名のコレクション。例: ["onClick","onDataChange"...].
f
formSizeAnchorフォームサイズを定義するために使用するオブジェクトの名前 (最小長さ: 1) (最小長さ: 1)4Dオブジェクトの名前
h
heightフォームの高さ最小値: 0
i
inheritedForm継承フォームを指定しますテーブルまたはプロジェクトフォームの名前 (文字列), フォームを定義する .json ファイルへの POSIX パス (文字列), またはフォームを定義するオブジェクト
inheritedFormTable継承フォームがテーブルに属していれば、そのテーブルテーブル名またはテーブル番号
m
markerBody詳細マーカーの位置最小値: 0
markerBreakブレークマーカーの位置最小値: 0
markerFooterフッターマーカーの位置最小値: 0
markerHeaderヘッダーマーカーの位置最小値 (整数): 0; 最小値 (整数配列): 0
memorizeGeometryフォームウィンドウが閉じられた時に、フォームパラメーターを記憶true, false
menuBarフォームと関連づけるメニューバー有効なメニューバーの名称
methodプロジェクトメソッド名存在するプロジェクトメソッドの名前
p
pagesページのコレクション (各ページはオブジェクトです)ページオブジェクト
pageFormatobject利用可能な印刷プロパティ
r
rightMargin水平マージン値 (ピクセル単位)最小値: 0
s
sharedフォームがサブフォームとして利用可能かを指定しますtrue, false
w
widthフォームの幅最小値: 0
windowMaxHeightフォームウィンドウの最大高さ最小値: 0
windowMaxWidthフォームウィンドウの最大幅最小値: 0
windowMinHeightフォームウィンドウの最小高さ最小値: 0
windowMinWidthフォームウィンドウの最小幅最小値: 0
windowSizingXフォームウィンドウの高さ固定"fixed", "variable"
windowSizingYフォームウィンドウの幅固定"fixed", "variable"
windowTitleフォームウィンドウのタイトルを指定しますフォームウィンドウの名前。
