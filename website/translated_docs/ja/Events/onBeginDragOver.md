## 説明

`On Begin Drag Over` フォームイベントは、ドラッグ可能なすべてのフォームオブジェクトで選択できます。 このイベントは、オブジェクトが [ドラッグ有効](FormObjects/properties_Action.md#ドラッグ有効) プロパティを持っているすべてのケースで生成されます。 このイベントは、ソースオブジェクトのメソッドまたはソースオブジェクトのフォームメソッドから呼び出すことができます。

> `On Begin Drag Over` イベントは、[`On Drag Over`](onDragOver.md) フォームイベントとは異なり、ドラッグアクションの **ソースオブジェクト** のコンテキスト内で呼び出されます。

`On Begin Drag Over` イベントは、ドラッグアクションの準備に役立ちます。 このイベントは以下のように使用できます:

- `APPEND DATA TO PASTEBOARD` コマンドを使って、ペーストボードにデータや署名を追加する。
- `SET DRAG ICON` コマンドを使って、ドラッグアクション中にカスタムアイコンを使用する。
- ドラッグされたオブジェクトのメソッドの $0 を使用して、ドラッグを受け入れるか拒否するかを設定する。
    - ドラッグ操作を受け入れることを示すには、ソースオブジェクトのメソッドが 0 (ゼロ) を返さなければなりません。そのため、`$0:=0` を実行する必要があります。
    - ドラッグ操作が拒否されたことを示すには、ソースオブジェクトのメソッドは -1 (マイナス1) を返さなければなりません。そのため、`$0:=-1` を実行する必要があります。
    - 結果が返されない場合は、ドラッグアクションが受け入れられたと 4D は判断します。

4D のデータは、イベントを呼び出す前にペーストボードに入れられます。 たとえば、**自動ドラッグ** アクションを使わずにドラッグした場合、イベントが呼び出された時点で、ドラッグされたテキストは既にペーストボードに入っています。
