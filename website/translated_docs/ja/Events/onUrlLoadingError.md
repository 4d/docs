title: On URL Loading Error
---

| コード | 呼び出し元                                     | 定義                 |
| --- | ----------------------------------------- | ------------------ |
| 50  | [Webエリア](FormObjects/webArea_overview.md) | URL をロード中にエラーが発生した |


## 説明

このイベントは、URL のロード中にエラーを検出すると生成されます。

`WA GET LAST URL ERROR` コマンドを使用して、エラーに関する情報を取得できます。
