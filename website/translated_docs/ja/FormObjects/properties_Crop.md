## 列

Sets the number of columns in a thumbnail table.

#### JSON 文法

| 名           | データタイプ  | とりうる値  |
|:----------- |:-------:| ------ |
| columnCount | integer | 最小値: 1 |


#### 対象オブジェクト

[Picture Button](pictureButton_overview.md) - [Button Grid](buttonGrid_overview.md) - [Picture Pop-up Menu](picturePopupMenu_overview.md)
