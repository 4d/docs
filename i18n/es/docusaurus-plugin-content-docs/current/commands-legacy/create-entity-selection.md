<!--REF #_command_.Create entity selection.Syntax-->**Create entity selection** ( *dsTable* {; *parametros*} ) -> Resultado<!-- END REF-->
<!--REF #_command_.Create entity selection.Params-->
| Parámetro | Tipo |  | Descripción |
| --- | --- | --- | --- |
| dsTable | Table | &srarr; | Tabla en la base 4D cuya selección actual se usará para crear la selección de entidad |
| parametros | Object | &srarr; | Opción de creación: contexto |
| Resultado | EntitySelection | &larr; | Selección de entidad que coincide con la clase de datos relacionada con la tabla dada |

<!-- END REF-->

#### Descripción 

<!--REF #_command_.Create entity selection.Summary-->**Documentación trasladada**

La documentación de este comando ha sido trasladada.<!-- END REF--> Visite *developer.4d.com* para ver la documentación actualizada.

El comando **Create entity selection** crea y devuelve una nueva selección de entidades alternable relacionada con la clase de datos que coincide con *dsTable*, de acuerdo con la selección actual de esta tabla.
