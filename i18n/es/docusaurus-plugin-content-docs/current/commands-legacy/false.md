#### Descripción 

<!--REF #_command_.False.Summary-->False devuelve el valor booleano Falso (False).<!-- END REF-->

#### Ejemplo 

El siguiente ejemplo asigna la variable *vbOpciones* a Falso:

```4d
 vbOpciones:=False
```
