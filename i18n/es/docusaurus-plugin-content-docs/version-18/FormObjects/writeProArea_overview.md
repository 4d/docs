title: Área 4D Write Pro
---

4D Write Pro ofrece a los usuarios de 4D una herramienta avanzada de procesamiento de textos, totalmente integrada a su base 4D. Con 4D Write Pro, puede escribir correos electrónicos y/o cartas preformateadas que contengan imágenes, una firma escaneada, texto formateado y marcadores de posición para variables dinámicas. También puede crear facturas o informes de forma dinámica, incluyendo texto e imágenes con formato.

![](../assets/en/FormObjects/writePro2.png)

## Utilizar las áreas 4D Write Pro

Las áreas 4D Write Pro están documentadas en el manual [4D Write Pro](https://doc.4d.com/4Dv17R6/4D/17-R6/4D-Write-Pro.100-4433851.fe.html).
