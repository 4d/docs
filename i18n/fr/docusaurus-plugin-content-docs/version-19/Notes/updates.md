## 4D v19.5

#### Points forts

- 4D v19.5 est certifié sur Windows 11 22H2
- 4D v19.5 est certifié sur macOS Ventura (macOS 13)
- La version de SpreadJS a été mise à jour (14.1.6).
- [**Liste des bugs corrigés**](https://bugs.4d.fr/fixedbugslist?version=19.5) : liste de tous les bugs qui ont été corrigés dans 4D v19.5.
