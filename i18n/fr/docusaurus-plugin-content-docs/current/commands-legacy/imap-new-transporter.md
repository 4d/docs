#### Description 

<!--REF #_command_.IMAP New transporter.Summary-->**Documentation déplacée**

La documentation de cette commande a été déplacée.<!-- END REF--> Consultez *developer.4d.com* pour obtenir une documentation à jour.

La commande **IMAP New transporter** configure une nouvelle connexion IMAP en fonction du paramètre *server* et retourne un nouvel objet *transporteur*. L'objet transporteur retourné sera alors généralement utilisé pour recevoir des e-mails.
