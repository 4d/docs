| Paramètre | Type |  | Description |
| --- | --- | --- | --- |

#### Description 

<!--REF #_command_.Open datastore.Summary-->**Documentation déplacée**

La documentation de cette section a été déplacée.<!-- END REF--> Consultez *developer.4d.com* pour obtenir la documentation mise à jour.

La commande **Open datastore** connecte l'application à la base de données 4D identifiée par le paramètre *infoConnexion* et retourne un objet datastore correspondant, associé à l'alias local *localID*.
