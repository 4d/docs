<!--REF #_command_.Macintosh option down.Syntax-->**Macintosh option down**  : Boolean<!-- END REF-->
<!--REF #_command_.Macintosh option down.Params-->
| Paramètre | Type |  | Description |
| --- | --- | --- | --- |
| Résultat | Boolean | &#8592; | Etat de la touche Option Macintosh ou Etat de la touche Alt Windows |

<!-- END REF-->

#### Description 

<!--REF #_command_.Macintosh option down.Summary-->**Macintosh option down** retourne Vrai si la touche **Option** Macintosh est enfoncée.<!-- END REF-->

**Note :** Lorsqu'elle est appelée sous Windows, la fonction **Macintosh option down** retourne Vrai si la touche **Alt** Windows est enfoncée.

#### Exemple 

Reportez-vous à l'exemple de la commande [Shift down](shift-down.md).
