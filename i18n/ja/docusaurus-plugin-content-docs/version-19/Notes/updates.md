## 4D v19.5

#### ハイライト

- 4D v19.5 is certified on Windows 11 22H2
- 4D v19.5 is certified on macOS Ventura (macOS 13)
- SpreadJS のバージョンが更新されました (14.1.6)。
- [**修正リスト**](https://bugs.4d.fr/fixedbugslist?version=19.5): 4D v19.5 で修正されたバグのリストです。
