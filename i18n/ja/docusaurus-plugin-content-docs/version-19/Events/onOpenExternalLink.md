## 詳細

このイベントは、`WA SET EXTERNAL LINKS FILTERS` コマンドで設定されたフィルターにより、URL のロードが Webエリアによってブロックされ、URL がカレントのシステムブラウザーで開かれると生成されます。

`WA Get last filtered URL` コマンドコマンドを使用してブロックされた URL を知ることができます。
