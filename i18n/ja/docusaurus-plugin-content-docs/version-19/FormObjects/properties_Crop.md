## 列

サムネールテーブルを構成する行数を指定します。

#### JSON 文法

| 名称          | データタイプ | とりうる値  |
|:----------- |:------:| ------ |
| columnCount |   整数   | 最小値: 1 |

#### 対象オブジェクト

[ピクチャーボタン](pictureButton_overview.md) - [ボタングリッド](buttonGrid_overview.md) - [ピクチャポップアップメニュー](picturePopupMenu_overview.md)
