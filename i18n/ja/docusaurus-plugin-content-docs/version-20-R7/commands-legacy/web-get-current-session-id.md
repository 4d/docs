#### 説明 

<!--REF #_command_.WEB Get Current Session ID.Summary-->**WEB Get Current Session ID**コマンドはカレントのWebリクエストのセッションIDを返します。<!-- END REF-->このIDは4Dが自動で生成します。 

このコマンドがWebセッション管理のコンテキストの外で呼び出されると、コマンドは空の文字列を返します。
