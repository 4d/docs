# OpenAIAPIResource

API リソースへのベ基本クラスです。

## プロパティ

| プロパティ     | 型                   | 説明                           |
| --------- | ------------------- | ---------------------------- |
| `_client` | [OpenAI](OpenAI.md) | OpenAI クライアントへのプライベートなバックリンク |

このクライアントを使用するとHTTPリクエストを行うことができます。
