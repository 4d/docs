# OpenAIModel

モデルの詳細。

https://platform.openai.com/docs/api-reference/models/object

## プロパティ

| プロパティ      | 型       | 説明                                                     |
| ---------- | ------- | ------------------------------------------------------ |
| `id`       | Text    | モデルの固有の識別子                                             |
| `created`  | Integer | モデルがいつ作成されたかを示すタイムスタンプ                                 |
| `object`   | Text    | オブジェクトの種類(例: model) |
| `owned_by` | Text    | オーナーの識別子(例: システムなど) |
