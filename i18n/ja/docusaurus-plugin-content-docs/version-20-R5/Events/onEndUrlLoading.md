title: On End URL Loading
---

| コード | 呼び出し元                                      | 定義                   |
| --- | ------------------------------------------ | -------------------- |
| 49  | [Web エリア](FormObjects/webArea_overview.md) | URL のすべてのリソースがロードされた |

## 説明

このイベントは、現在の URL のすべてのリソースがロードが完了すると生成されます。 `WA Get current URL` コマンドを使用して、ロードされた URL を知ることができます。
