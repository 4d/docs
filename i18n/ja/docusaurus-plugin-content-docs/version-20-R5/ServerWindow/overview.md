title: 4D Server 管理ウィンドウ
---

Windows または macOS上で、4D Server をインターフェースありで起動すると、グラフィカルな管理ウィンドウが利用でき、公開された 4Dアプリケーション用に多くの解析・制御ツールを提供します。 To display the 4D Server Administration window for the opened project, select the **Window > Administration** menu item, or press **Ctrl+U**.

> 管理ウィンドウはリモートの 4D からもアクセス可能です。 For more information about this point, please refer to the [Administration from Remote Machines](remote-admin.md) page.
