## 横線カラー

リストボックス内の横線の色を指定します (デフォルトはグレー)。

#### JSON 文法

| 名称                   | データタイプ | とりうる値                                |
| -------------------- | ------ | ------------------------------------ |
| horizontalLineStroke | color  | 任意の css値; "transparent"; "automatic" |

#### 対象オブジェクト

[List Box](listbox_overview.md)
