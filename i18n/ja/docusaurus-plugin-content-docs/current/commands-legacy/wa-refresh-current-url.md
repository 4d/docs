<!--REF #_command_.WA REFRESH CURRENT URL.Syntax-->**WA REFRESH CURRENT URL** ( {* ;} *object* )<!-- END REF-->
<!--REF #_command_.WA REFRESH CURRENT URL.Params-->
| 引数 | 型 |  | 説明 |
| --- | --- | --- | --- |
| * | 演算子 | &rarr; | 指定時, objectはオブジェクト名 (文字列) 省略時, objectは変数 |
| object | any | &rarr; | オブジェクト名 (* 指定時) または 変数 (* 省略時) |

<!-- END REF-->

#### 説明 

<!--REF #_command_.WA REFRESH CURRENT URL.Summary-->WA REFRESH CURRENT URL コマンドは、*\** と *object* 引数で指定したWebエリアに現在表示されているURLを再読み込みします。<!-- END REF-->
