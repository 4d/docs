<!--REF #_command_.Tan.Syntax-->**Tan** ( *number* ) -> 戻り値<!-- END REF-->
<!--REF #_command_.Tan.Params-->
| 引数 | 型 |  | 説明 |
| --- | --- | --- | --- |
| number | Real | &#8594;  | タンジェントを求めるラジアン値 |
| 戻り値 | Real | &#8592; | タンジェント値 |

<!-- END REF-->

#### 説明 

<!--REF #_command_.Tan.Summary-->Tan は*number*の正接値を返します。<!-- END REF-->*number*はラジアンで指定します。

**Note:** 4DではPi、Degree、Radianという定数があらかじめ定義されています。Piはパイの数値 (3.14159...) を返し、Degreeはラジアンで表わされた1度 (0.01745...) を、Radianは度数で表わされた1ラジアン (57.29577...) を返します。
