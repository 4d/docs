## Descrição

O evento `On Losing Focus`, juntamente com o evento [`On Getting Focus`](onGettingFocus.md), são usados para detectar e tratar a mudança de foco para objetos [focusable](FormObjects/properties_Entry.md#focusable).

Com [objetos de subformulário] (FormObjects/subform_overview.md), esse evento é gerado no método do objeto de subformulário quando ele é verificado. É enviado para o método de formulário do subformulário, o que significa, por exemplo, que você pode gerenciar a exibição dos botões de navegação no subformulário de acordo com o foco. Note-se que os objetos de subformulário podem ter eles próprios o foco.
