#### Descrição 

<!--REF #_command_.POP3 New transporter.Summary-->**Documentação movida**

A documentação para esse comando foi movida.<!-- END REF--> Visite *developer.4d.com* para documentação atualizada.

O comando **POP3 New transporter** configura uma nova conexão POP3 de acordo com o parâmetro *servidor* e retorna um novo objeto transportador. O objeto retornado *transportador* será usado para receber emails.
