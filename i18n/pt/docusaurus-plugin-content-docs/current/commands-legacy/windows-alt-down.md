<!--REF #_command_.Windows Alt down.Syntax-->**Windows Alt down**  : Boolean<!-- END REF-->
<!--REF #_command_.Windows Alt down.Params-->
| Parâmetro | Tipo |  | Descrição |
| --- | --- | --- | --- |
| Resultado | Boolean | &#8592; | Estado da tecla Alt em windows (ou tecla Opção em Macintosh) |

<!-- END REF-->

#### Descrição 

Windows Alt down retorna TRUE se a tecla Alt Windows estiver pressionada.  

**Nota**: quando chamado numa plataforma Macintosh, Windows Alt down retorna TRUE se a tecla Macintosh Opção estiver pressionada.
