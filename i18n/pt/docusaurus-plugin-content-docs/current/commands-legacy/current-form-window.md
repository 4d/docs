<!--REF #_command_.Current form window.Params-->
| Parâmetro | Tipo |  | Descrição |
| --- | --- | --- | --- |
| Resultado | Integer | &#8592; | Número de referência da janela do formulário atual |

<!-- END REF-->

*Esse comando não é seguro para thread e não pode ser usado em código adequado.*


#### Descrição 

<!--REF #_command_.Current form window.Summary-->O comando Current form window devolve a referência da janela do formulário atual.<!-- END REF--> Se nenhuma janela tiver sido definida para o formulário atual, o comando devolve 0.  

A janela do formulário atual pode ser gerada automaticamente por um comando tal como ADD RECORD, depois de uma ação de usuário ou utilizando os comandos [Open window](open-window.md) ou [Open form window](open-form-window.md).
