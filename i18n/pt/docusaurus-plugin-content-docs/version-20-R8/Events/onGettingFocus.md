## Descrição

O evento `On Getting Focus`, juntamente com o evento [`On losing Focus`](onLosingFocus.md), são usados para detectar e tratar a mudança de foco para objetos [focusable](FormObjects/properties_Entry.md#focusable).

Con los [objetos subformulario](FormObjects/subform_overview.md), este evento se genera en el método del objeto subformulario cuando se verifica. É enviado para o método de formulário do subformulário, o que significa, por exemplo, que você pode gerenciar a exibição dos botões de navegação no subformulário conforme o foco. Note-se que os objetos de subformulário podem ter eles próprios o foco.
