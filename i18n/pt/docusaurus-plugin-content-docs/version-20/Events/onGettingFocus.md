## Descrição

Os eventos `On Getting Focus` e [`On loss Focus`](onLosingFocus.md) são usados para detectar e gerenciar mudanças de foco para objetos [focalizáveis](FormObjects/properties_Entry.md#focusable).

Com os [objetos subformulário](FormObjects/subform_overview.md), este evento é gerado no método do objeto subformulário quando ele é marcado. É enviado para o método de formulário do subformulário, o que significa, por exemplo, que você pode gerenciar a exibição dos botões de navegação no subformulário conforme o foco. Note-se que os objetos de subformulário podem ter eles próprios o foco.
