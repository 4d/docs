| Code | Pode ser chamado por | Definição                                                                           |
| ---- | -------------------- | ----------------------------------------------------------------------------------- |
| 40   | Formulário           | Durante a entrada do usuário na lista, um registro é carregado e um campo é editado |


## Descrição

O evento `On Load Record` só pode ser usado no contexto de um **formulário de saída**. É acionado durante a entrada de dados na lista, depois que um registro é destacado e um campo é alterado para o modo de edição.

> Este evento não pode ser selecionado para formulários projeto, está disponível apenas com **formulários tabela**.
