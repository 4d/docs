Administración Web

Herramientas web 4D para administrar y supervisar sus aplicaciones.

📄️ WebAdmin

Un componente de servidor web integrado, llamado WebAdmin, es utilizado por 4D y 4D Server para dar un acceso web seguro a funciones de gestión específicas como el Explorador de datos. Puede conectarse local o remotamente a este servidor web desde un navegador o cualquier aplicación web y acceder a la aplicación 4D asociada.

📄️ Explorador de datos

El Explorador de datos ofrece una interfaz web para ver, consultar y editar los datos del almacén de datos de su proyecto. Utilizando esta herramienta, puede navegar fácilmente entre todas sus entidades y buscar, ordenar o filtrar los valores de los atributos. Le ayuda a identificar rápidamente los problemas en cualquier paso del proceso de desarrollo, así como a controlar y editar los datos.
