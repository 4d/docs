|[**DELETE FROM LIST** ( {* ;} *list* ; itemRef | * {; *} )](../../commands-legacy/delete-from-list)<br/>The **DELETE FROM LIST** command deletes the item designated by the *itemRef* parameter of the list whose reference number or object name is passed in *list*.|
|[**Find in list** ( {* ;} *list* ; *value* ; *scope* {; *itemsArray* {; *}} ) : Integer](../../commands-legacy/find-in-list)<br/>The **Find in list** command returns the position or reference of the first item of the *list* that is equivalent to the string passed in *value*.|

|[**Get list item font** ( {* ;} *list* ; itemRef | * ) : Text](../../commands-legacy/get-list-item-font)<br/>The **Get list item font** command returns the current character font name of the item specified by the *itemRef* parameter of the list whose reference number or object name is passed in *list*.|
|[**GET LIST ITEM ICON** ( {* ;} *list* ; itemRef | * ; *icon* )](../../commands-legacy/get-list-item-icon)<br/>The **GET LIST ITEM ICON** command returns, in *icon*, the icon associated with the item whose reference number is passed in *itemRef* in the list whose reference number or object name is passed in *list*.|
|[**GET LIST ITEM PARAMETER** ( {* ;} *list* ; itemRef | * ; *selector* ; *value* )](../../commands-legacy/get-list-item-parameter)<br/>The **GET LIST ITEM PARAMETER** command is used to find out the current *value* of the *selector* parameter for the *itemRef* item of the hierarchical list whose reference or object name is passed in the *list* parameter.|
|[**GET LIST ITEM PARAMETER ARRAYS** ( {* ;} *list* ; itemRef | * ; *arrSelection* {; *arrValues*} )](../../commands-legacy/get-list-item-parameter-arrays)<br/>The **GET LIST ITEM PARAMETER ARRAYS** command lets you retrieve all the parameters in a single call (as well as, optionally, their values) that are associated with the *itemRef* item in the hierarchical list whose reference or object name is passed in the *list* parameter.|
|[**GET LIST ITEM PROPERTIES** ( {* ;} *list* ; itemRef | * ; *enterable* {; *styles* {; *icon* {; *color*}}} )](../../commands-legacy/get-list-item-properties)<br/>The **GET LIST ITEM PROPERTIES** command returns the properties of the item designated by the *itemRef* parameterwithin the list whose list reference number or object name is passed in *list*.|
|[**GET LIST PROPERTIES** ( *list* ; *appearance* {; *icon* {; *lineHeight* {; *doubleClick* {; *multiSelections* {; *editable*}}}}} )](../../commands-legacy/get-list-properties)<br/>The **GET LIST PROPERTIES** command returns information about the list whose reference number you pass in *list*.|
|[**INSERT IN LIST** ( {* ;} *list* ; beforeItemRef | * ; *itemText* ; *itemRef* {; *sublist* ; *expanded*} )](../../commands-legacy/insert-in-list)<br/>The INSERT IN LIST command inserts the item designated by the *itemRef* parameter in the list whose reference number or object name you pass in *list*.|
|[**Is a list** ( *list* ) : Boolean](../../commands-legacy/is-a-list)<br/>The Is a list command returns TRUE if the value you pass in *list* is a valid reference to a hierarchical list.|
|[**List item parent** ( {* ;} *list* ; itemRef | * ) : Integer](../../commands-legacy/list-item-parent)<br/>The **List item parent** command returns the item reference number of a parent item.|
|[**List item position** ( {* ;} *list* ; *itemRef* ) : Integer](../../commands-legacy/list-item-position)<br/>The **List item position** command returns the position of the item whose item reference number is passed in *itemRef*, within the list whose list reference number or object name is passed in *list*.|
|[**LIST OF CHOICE LISTS** ( *numsArray* ; *namesArray* )](../../commands-legacy/list-of-choice-lists)<br/>The LIST OF CHOICE LISTS command returns, in the synchronized *numsArr* and *namesArr* arrays, the numbers and names of the choice lists defined by the list editor in Design mode.|
|[**Load list** ( *listName* ) : ListRef](../../commands-legacy/load-list)<br/>**Load list** creates a new hierarchical list whose contents are copied from the list and whose name you pass in *listName*.|
|[**New list**  : ListRef](../../commands-legacy/new-list)<br/>**New list** creates a new, empty hierarchical list in memory and returns its unique list reference number.|
|[**SAVE LIST** ( *list* ; *listName* )](../../commands-legacy/save-list)<br/>The SAVE LIST command saves the list whose reference number you pass in *list*, within the Design environment List Editor, under the name you pass in *listName*.|
|[**SELECT LIST ITEMS BY POSITION** ( {* ;} *list* ; *itemPos* {; *positionsArray*} )](../../commands-legacy/select-list-items-by-position)<br/>The **SELECT LIST ITEMS BY POSITION** command selects the item(s) whose position is passed in *itemPos* and, optionally, in *positionsArray* within the list whose reference number or object name is passed in *list*.|
|[**SELECT LIST ITEMS BY REFERENCE** ( *list* ; *itemRef* {; *refArray*} )](../../commands-legacy/select-list-items-by-reference)<br/>The **SELECT LIST ITEMS BY REFERENCE** command selects the item(s) whose item reference number is passed in *itemRef* and, optionally, in *refArray*, within the list whose reference number is passed in *list*.|
|[**Selected list items** ( {* ;} *list* {; *itemsArray* {; *}} )  : Integer](../../commands-legacy/selected-list-items)<br/>The **Selected list items** command returns the position or reference of the selected item in the list whose reference number or object name you pass in *list*.|

|[**SET LIST ITEM ICON** ( {* ;} *list* ; itemRef | * ; *icon* )](../../commands-legacy/set-list-item-icon)<br/>The **SET LIST ITEM ICON** command modifies the icon associated with the item specified by the *itemRef* parameter of the list whose reference number or object name is passed in *list*.|
|[**SET LIST ITEM PARAMETER** ( {* ;} *list* ; itemRef | * ; *selector* ; *value* )](../../commands-legacy/set-list-item-parameter)<br/>The **SET LIST ITEM PARAMETER** command modifies the *selector* parameter for the *itemRef* item of the hierarchical list whose reference or object name is passed in the *list* parameter.|
|[**SET LIST ITEM PROPERTIES** ( {* ;} *list* ; itemRef | * ; *enterable* ; *styles* {; *icon* {; *color*}} )](../../commands-legacy/set-list-item-properties)<br/>The **SET LIST ITEM PROPERTIES** command modifies the item designated by the *itemRef* parameter within the list whose reference number or object name is passed in *list*.|
