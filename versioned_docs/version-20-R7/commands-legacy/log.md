#### Description 

<!--REF #_command_.Log.Summary-->Log returns the natural (Napierian) log of *number*.<!-- END REF--> Log is the inverse function of [Exp](exp.md).

**Note:** 4D provides the predefined constant *e number* (2.71828...).
