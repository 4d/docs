Área 4D Write Pro

4D Write Pro ofrece a los usuarios de 4D una herramienta avanzada de procesamiento de textos, totalmente integrada a su base 4D. Using 4D Write Pro, you can write pre-formatted emails and/or letters containing images, a scanned signature, formatted text and placeholders for dynamic variables. You can also create invoices or reports dynamically, including formatted text and images.

Using 4D Write Pro areas

4D Write Pro areas are documented in the 4D Write Pro Reference manual.
